# Content Calendar — Only Once Share (ooshare.io)

> 12-month content roadmap organized by priority and type.
> Created: March 2026

---

## Content Principles

1. **Technical depth over marketing fluff** — Target developers and IT professionals who value accuracy
2. **Answer real questions** — Every post should target a query people actually search for
3. **Show, don't tell** — Include code examples, screenshots, architecture diagrams
4. **Multilingual advantage** — Translate high-priority content into all 6 supported languages
5. **Update regularly** — Comparison and "best of" content must stay current

## Content Quality Checklist

Every piece of content should:

- [ ] Target a specific keyword with documented search volume
- [ ] Include a clear, keyword-optimized title (< 60 characters)
- [ ] Have a meta description (< 160 characters) with target keyword
- [ ] Use proper heading hierarchy (H1 → H2 → H3)
- [ ] Include at least one internal link to another ooshare.io page
- [ ] Include relevant schema markup (Article, HowTo, FAQPage)
- [ ] Be factually accurate with sources cited
- [ ] Include a clear CTA (try ooshare.io, self-host, share securely)
- [ ] Be formatted for AI citability (clear definitions, structured lists)
- [ ] Pass readability check (grade 8-10 reading level)
