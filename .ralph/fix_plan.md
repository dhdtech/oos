# Ralph Fix Plan

## Architecture Overview
**Only Once Share (OOS)** - A one-time secret sharing service.
- **Backend**: Python/Flask via connexion (OpenAPI-first), Fernet encryption, Redis storage
- **Frontend**: Vue.js (ui/oos/)
- **Infra**: Docker + docker-compose, GitHub Actions CI/CD
- **Core Flow**: POST /secret/ → encrypt + store in Redis (24h TTL) → return secret_id + recovery_key → GET /secret/{id}?xnxsoss={key} → decrypt + delete from Redis
